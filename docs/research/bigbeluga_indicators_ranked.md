## Note Tecniche

**NautilusTrader Mapping Challenges**:
- FVG detection → Custom implementation (no native)
- Volume Profile → Custom (no native VP in Nautilus)
- Delta calculation → Tick-level data required for accuracy
- Liquidation zones → Pivot + ATR (già implementato)

**Data Requirements**:
- Volume Profile indicators → Need tick/bar volume
- Delta indicators → Ideally need bid/ask data (or use close vs open heuristic)
- Liquidation → Works with OHLCV
